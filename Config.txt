Details:
- Interrupt: PORT1_VECTOR

üîπ 5. SOS Push Button (S1 on LaunchPad)
| Feature   | MSP430 Pin | Direction | Type                |
| --------- | ---------- | --------- | ------------------- |
| S1 Button | **P2.1**   | Input     | Digital + Interrupt |


Details:
- Interrupt: PORT2_VECTOR
- Active LOW
- Emergency trigger (1 sec buzzer)

üîπ 6. Flame Sensor (Fire Detection)
| Feature    | MSP430 Pin | Direction | Type                |
| ---------- | ---------- | --------- | ------------------- |
| Flame OUT  | **P2.7**   | Input     | Digital + Interrupt |


Details:
- Interrupt: PORT2_VECTOR
- Trigger: High ‚Üí Low (flame detected)
- Highest priority: Opens door automatically
- Sets fire_override_active flag

üîπ 7. MQ-6 Gas Sensor (Analog)
| Feature            | MSP430 Pin    | Direction | Type          |
| ------------------ | ------------- | --------- | ------------- |
| MQ Analog OUT (AO) | **P6.0 (A0)** | Input     | ADC (12-bit)  |
| MQ VCC             | 5 V           | Power     | Heater supply |
| MQ GND             | GND           | Power     | Common ground |


Details:
- Sampling via ADC12
- Periodic check via Timer A0
- Threshold: 1800 (adjustable)
- Displays value on LCD Line 1

üîπ 8. Timer (Gas Sensor Polling)
| Feature  | Module | Purpose                 |
| -------- | ------ | ----------------------- |
| Timer A0 | TA0    | Periodic MQ ADC trigger |


Details:
- Source: ACLK (32 kHz)
- Interval: ~0.5 sec (16384 counts)
- ISR: TIMER0_A0_VECTOR
- Sets gas_check_flag

üîπ 9. Servo Motor (Door Lock - SG90)
| Feature       | MSP430 Pin | Direction | Type      |
| ------------- | ---------- | --------- | --------- |
| Servo Signal  | **P2.0**   | Output    | PWM (TA1.1) |
| Servo VCC     | 5 V        | Power     | External recommended |
| Servo GND     | GND        | Power     | Common ground |


Details:
- Timer: A1 (TA1)
- PWM Frequency: 50 Hz (20ms period)
- Open (90¬∞): 1500-2000 ¬µs pulse
- Close (0¬∞): 1000 ¬µs pulse
- Module: TA1.1 on P2.0
- IMPORTANT: Needs external 5V power (500mA+)

üîπ 10. LCD Display (16√ó2 I2C)
| Feature | MSP430 Pin | Direction     | Protocol      |
| ------- | ---------- | ------------- | ------------- |
| LCD SDA | **P3.0**   | Bidirectional | I2C           |
| LCD SCL | **P3.1**   | Output        | I2C           |
| LCD VCC | 5 V        | Power         | With backpack |
| LCD GND | GND        | Power         | Common        |


Details:
- I2C module: UCB0
- Address: 0x27 (or 0x3F)
- Clock: SMCLK @ 100kHz
- Displays:
  - Line 1: Gas sensor value
  - Line 2: Password entry / status

üîπ 11. UART Communication (Password Entry)
| Feature  | MSP430 Pin | Direction | Protocol |
| -------- | ---------- | --------- | -------- |
| UART TX  | **P4.4**   | Output    | UART     |
| UART RX  | **P4.5**   | Input     | UART     |


Details:
- Module: USCI_A1
- Baud Rate: 9600 bps
- Clock: SMCLK (1 MHz) with 16x oversampling
- ISR: USCI_A1_VECTOR
- Password: "1234" (4 digits)
- Validates only '0'-'9' characters
- On correct password: Unlocks door + Green LED blink
